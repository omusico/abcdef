第2天 三亚 蜈支洲岛（推荐行程）

  • 早上

    享用【早餐】，餐费已含，用餐时间约1小时, 用餐地点 酒店内，温馨提醒：请注意酒店早餐使用时间。

  • 上午

    可前往推荐景点蜈支洲岛度假中心（费用自理158人民币，此景点有门票景点费用，游览时间约3小时）在蜈支洲岛，可寻访妈祖庙古迹，也可潜入水中和海底生物亲密接触，或面朝大海等一轮新日，是宁静、清丽的度假天堂。

  • 下午
    自由活动

    下岛后返回酒店，可选择到红树林酒店内品尝金花茶。傍晚时分，到浪漫迷人的【亚龙湾沙滩】,名符其实的东方夏威夷，躺在舒适躺椅上,听着海浪，吹着海风，让您心旷神怡，留恋往返，您还可尽情漫步在洁白的沙滩上。

第3天 三亚 亚龙湾天堂森林公园（推荐行程）

  • 早上
    早餐

    用餐时间 1小时

    用餐地点：酒店内

  • 上午

    可前往推荐景点亚龙湾热带天堂森林公园（费用自理135人民币，此景点有门票景点费用，游览时间约3小时）亚龙湾热带天堂森林公园是离三亚市的天然森林氧吧，因为《非诚勿扰2》取景于此，而名声大噪。行走于海天之间、雨林之上，堪称海南岛的“香格里拉”，“体会世界稀有的热带雨林景观，登龙山观龙湾品味“东龙西凤”文化，赏天然巨佛，祈幸福安康,重温笑笑与秦奋的冯氏浪漫爱情喜剧。

  • 下午
    自由活动

    活动时间： 约3小时

    返回酒店后小憩一会，红树林的礁湖泳池将让您游的畅快淋漓，酒店每日都有不同的活动日程，可挑选几个参加，让轻松的价格更加精彩。

  • 入住 三亚亚龙湾红树林度假酒店
